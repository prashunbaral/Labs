# 🧪 LFI Security Lab

Welcome to the **Local File Inclusion (LFI)** lab! This project is designed to help you understand and exploit common LFI vulnerabilities in modern web applications.

## 🚀 Getting Started

### 1. Prerequisites
Ensure you have **Node.js** and **npm** installed on your machine.
- [Download Node.js](https://nodejs.org/)

### 2. Setup the Lab
Clone this repository and navigate to the project folder in your terminal:

\`\`\`bash
# Install the necessary dependencies
npm install
\`\`\`

### 3. Run the Application
Start the lab server by running:

\`\`\`bash
npm start
\`\`\`

### 4. Access the Lab
Once the server is running, open your web browser and go to:
👉 **[http://localhost:3000](http://localhost:3000)**

---

## 🎯 Lab Objectives
- **Register**: Enter your **Student ID** on the home page to initialize your unique environment.
- **Find Flags**: There are **three flags** hidden within the application. Each level requires a different LFI technique.
- **Format**: Flags are in the format \`ETHICALHCK{...}\`.
- **Submit**: Use the **Submit Flags** page within the app to verify your solutions.

## 📂 Challenges
1. **Level 1**: Basic path traversal in the legacy support portal.
2. **Level 2**: Filter bypass in the asset optimizer.
3. **Level 3**: Advanced URL-encoded traversal in the profile sync service.
